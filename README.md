## 🚀 Inicio rápido

### Opción 1: GitHub Pages (Recomendado)

1. Ve a **Settings** → **Pages**
2. Selecciona **Deploy from a branch**
3. Elige rama `main` y carpeta `/root`
4. Tu sitio estará disponible en: `https://dealturaingenieria-cmyk.github.io/isa-ingenieria`

### Opción 2: Dominio personalizado

1. Compra un dominio (ej: isaingenieria.com)
2. Configura los DNS de tu proveedor:
   - **A Records:**
     - 185.199.108.153
     - 185.199.109.153
     - 185.199.110.153
     - 185.199.111.153
   - **O CNAME:**
     - dealturaingenieria-cmyk.github.io

3. En **Settings** → **Pages** → **Custom domain**, escribe tu dominio
4. Marca **Enforce HTTPS**
5. ¡Listo! Tu sitio estará disponible en tu dominio personalizado
